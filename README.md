# LainaaMut (BorrowMe)   :movie_camera: :game_die: :boat:  :guitar:
AirBnB for items 

## Introduction
Add items you own and make reservations on friends items.

### Features  :white_check_mark:
- add item
- list items
- make reservation
- list reservations for item
- list all reservations
- delete item
